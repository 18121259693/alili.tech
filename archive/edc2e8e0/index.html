    【前端芝士树】如何对元素块实现垂直居中？

    1/利用Flex布局来实现极速居中

    The HTML

    <div class="container">
    <!--// Any content in here will be centered.-->
    <img src="fireworks.jpg" alt="fireworks">
</div>

    The CSS

    .container{
    display: flex;
    justify-content: center;
    align-items: center;
}

    2/利用CSS和JS实现固定长宽容器的极速居中

    The HTML

    .className{
    margin:0 auto;
    width:200px;
    height:200px;
}

    Horizontal and vertical centering with CSS

    .className{
    width:300px;
    height:200px;
    position:absolute;
    left:50%;
    top:50%;
    margin:-100px 0 0 -150px;
}

    Horizontal and vertical centering with jQuery

    $(window).resize(function(){

    $('.className').css({
        position:'absolute',
        left: ($(window).width() - $('.className').outerWidth())/2,
        top: ($(window).height() - $('.className').outerHeight())/2
    });

});

// To initially run the function:
$(window).resize();

    3/已知子元素宽高，利用margin:auto实现fixed元素的居中

    The CSS

    div{
   position: fixed;
   margin: auto;
   top: 0;
   right: 0;
   left: 0;
   bottom: 0;

   width: 200px;
   height: 100px;
}
