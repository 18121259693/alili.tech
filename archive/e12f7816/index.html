前端最受欢迎的 UI 框架

2018-11-11
0

前端最受欢迎的 UI 框架

  1. bootstrap: 由 twitter 出品的 UI 框架。经过多年的发展，社区构建出了一个强大的生态，以及各种各样的插件，基本上你所需要的都能找到。框架主要以 css 为主，js 为辅，js 组件部分依赖 jquery。也有针对 react 的适配 react-bootstrap 和 vue 的适配 bootstrap-vue。
  2. Semantic-UI: 语义化的 UI 框架。与 bootstrap 类似，主要以 css 为主，js 为辅，js 组件部分依赖 jquery。但与 bootstrap 不同的是，它的命名采用语义化，而非连字符 - （如：<button class="ui primary button">Primary</button>, <div class="ui small image"><img src="image.png"></div>）。也有针对 react 的适配 Semantic-UI-React。
  3. materialize: Google 材质设计（Material Design）for web.
  4. material-ui: Google 材质设计（Material Design）for react.
  5. ant-design: 蚂蚁金服出品的一套 UI 框架（for react）.
  6. bulma: 另一套类似 bootstrap 和 Semantic-UI 的 UI 框架。
  7. element: 饿了么官方出品的一套 UI 框架（for vue）.
  8. iview: 另一套 UI 框架（for vue）.
