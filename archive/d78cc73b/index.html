    【开发环境】开发者，新的mac 笔记本推荐配置环境

    2018-11-15
    0

    本地添加ssh

    ssh-keygen -t rsa -C "wsd312@163.com"

    安装 homebrew

    /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

    下载 ss

    mac 版本
https://github.com/shadowsocks/ShadowsocksX-NG/releases

    邮箱

    百度和沪江都是outlook 微软的邮箱web系统

    • 邮箱大师 163出品就可以搞定
    • 自带的邮件系统
    • 手机使用的也是同样的客户端
    之前有的公司使用的是腾讯邮箱，可以关联小程序 和公众号，有邮件会通知的

    开发工具

    安装homebrew

    iterm2

    设置默认字体 18

    安装zshell 配置

    管理zshell 插件

    https://github.com/tarjoilija...

    git clone https://github.com/tarjoilija/zgen.git "${HOME}/.zgen" && source "${HOME}/.zgen/zgen.zsh"

# iterm bar show num
zgen load tysonwolker/iterm-tab-colors

    配置主题 并且配置字体 和颜色

    ZSH_THEME="agnoster"

    配置插件

    参考 https://github.com/unixorn/aw...

    vim  ~/.zshrc
找到 plugins 添加下面插件 
plugins=(
  git
  extract
  sublime
)
source  ~/.zshrc

    配置自动跳转

    brew install autojump
    这个插件，只要你访问过的文件夹，可以 j filename 快速进入

    编辑器

    sublime

    atom

    atom插件

    自己写的插件，写文档可以上传图片到七牛
    markdown-up-img

    vscode 这三个配合使用

    终端快捷启动 vscode

    或者直接添加 zsh vscode 插件 上面讲到了

    vim ~/.zshrc

alias code='/Applications/Visual\ Studio\ Code.app/Contents/Resources/app/bin/code'

source ~/.zshrc

    安装node

    brew install node

    配置淘宝镜像

    alias cnpm="npm --registry=https://registry.npm.taobao.org \
--cache=$HOME/.npm/.cache/cnpm \
--disturl=https://npm.taobao.org/dist \
--userconfig=$HOME/.cnpmrc"

    mysql 下载

    安装方式

    brew install mysql

    结果

    We've installed your MySQL database without a root password. To secure it run:
    mysql_secure_installation

MySQL is configured to only allow connections from localhost by default

To connect run:
    mysql -uroot

To have launchd start mysql now and restart at login:
  brew services start mysql
Or, if you don't want/need a background service you can just run:
  mysql.server start

🍺  /usr/local/Cellar/mysql/5.7.22: 317 files, 234.2MB
